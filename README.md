# patient_portal

Une application mobile Flutter permettant à un patient de consulter son carnet de suivi médical.

## 📱 Fonctionnalités

### 🏠 Écran d'accueil (/)

### 📅 Écran Rendez-vous (/rendez-vous)

### 🗂️ Écran Dossier (/dossier)

### 💡 Écran Recommandations (/recommandations)
